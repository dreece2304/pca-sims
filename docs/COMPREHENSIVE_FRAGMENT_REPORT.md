### **🔗 Chemical Transformation Series Identified**

#### **1. Carbonyl Cascade Formation** (Thermodynamic Stabilization)
```
HO-CH₂-C≡C-CH₂-OH → C₄H₆O₂⁻ (intact diol)
       ↓ e-beam radical chemistry
    C₄HO⁻ (+119%) → C₃HO⁻ (+69%) → C₂HO⁻ (+52%) → COOH⁻ (+98%)
```
- **Mechanism**: Progressive carbonyl formation for C=O bond stabilization
- **Evidence**: All carbonyls show dose-dependent increases
- **Thermodynamics**: C=O bonds provide ~180 kJ/mol stabilization energy

#### **2. Aromatic Network Formation** (Maximum Stability)
```
Crosslinked carbons → C₆H⁻ aromatic rings (+154%)
```
- **Mechanism**: Conjugated π-system formation for ultimate stability
- **Evidence**: C₆H⁻ shows highest percentage increase (154%)
- **Thermodynamics**: Aromatic delocalization = highest stability state

#### **3. Hydrogen Redistribution Chemistry**
```
Initial crosslinking: polymer-H → H⁻ (radical chemistry)
High-dose carbonization: H⁻ → stabilized C-H bonds
```
- **Mechanism**: H⁻ production during radical formation, consumption in stabilization
- **Evidence**: H⁻ decreases (-28%) from SQ2 to SQ5
- **Role**: Central to all transformation processes
